## TransitionsKit
A set of predefined transition components build on top of `react-transition-gorup`

## Usage example
##### Fade:
```jsx
<Fade in={open}>
    <p>Content</p>
</Fade>
```

##### Collapse:
```jsx
<Collapse in={open} unmountOnExit>
    <p>Content</p>
</Collapse>
```

##### Blur:
```jsx 

<Blur in={open} timout={500}>
    <p>Content</p>
</Blur>
```
